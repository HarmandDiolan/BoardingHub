# BoardingHub

Commands to run:

After cloning from github

>composer install <br />
>cp .env.example .env <br />

edit the .env

After editing .env

>php artisan key:generate <br />
>php artisan migrate <br />
>npm install <br />
>npm run dev <br />
>php artisan serve <br />

localhost:8000 to view the system
